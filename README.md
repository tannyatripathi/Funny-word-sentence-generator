# 😂 Funny Sentence Generator

## Overview
A simple web application that generates random humorous sentences by dynamically combining different words. Built to demonstrate basic JavaScript logic and DOM manipulation.

## Features
- Generates random funny sentences on button click
- Uses word replacement logic (subjects, verbs, objects)
- Interactive and lightweight UI
- Instant output without page reload

## Tech Stack
- HTML
- CSS
- JavaScript (Basic)

## How It Works
The application randomly selects words from predefined arrays (subjects, verbs, objects) and combines them to form a sentence.

Example:
> "The robot eats the pizza 😂"

## Purpose
This project was created to practice:
- JavaScript fundamentals
- DOM manipulation
- Event handling
- Basic UI interaction

## Future Improvements
- Add user input customization
- Add more word categories
- Improve UI design
- Add sound effects for fun interaction

## How to Run
1. Download or clone the repository
2. Open `index.html` in a browser
